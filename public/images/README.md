# Image Assets

All images below are **solid-navy PNG placeholders** generated by
`scripts/gen-placeholders.mjs`. They exist so the layout renders correctly and
every `next/image` has a valid, sized `src`. To ship real photography, just
**overwrite the file at the same path** (keep the filename) — no code changes
needed. If you prefer `.jpg`, change the path in `src/lib/site-config.ts`.

> Tip: re-run `node scripts/gen-placeholders.mjs` anytime to regenerate the
> placeholders.

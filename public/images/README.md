# Image Assets

All raster images below are **solid-navy JPEG placeholders** generated by
`scripts/gen-placeholders.mjs` (pure JS via `jpeg-js`, no native deps). They
exist so the layout renders correctly and every `next/image` has a valid, sized
`src`. To ship real photography, just **overwrite the file at the same path**
(keep the filename) — no code changes needed.

> Regenerate placeholders anytime: `npm run gen:placeholders`

## ⚠️ Compression note (important for performance)

`next.config.mjs` sets `images.unoptimized: true` because Cloudflare Pages
doesn't run Next's default optimizer. That means **whatever bytes you drop in are
served as-is** — so compress real photos before adding them (target ~150–300 KB
for cards, ~300–500 KB for the hero), or wire up Cloudflare Image Resizing.
Otherwise large originals will hurt load time and Lighthouse.
